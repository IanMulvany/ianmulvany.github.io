Disagreement Patterns

When two annotators label the same feedback item differently, each difference is a “disagreement pattern” (e.g. neutral→negative means Annotator A labeled it neutral, Annotator B labeled it negative). Bars show how often each pattern occurs.

Annotator Bias Summary

IM tends to label more items as neutral and positive.

NN tends to label significantly more items as negative (46% vs 17%).

JW and HM label the same items as IM/NN; their labels are included in the consensus via majority vote.

The consensus approach uses the more critical label when annotators disagree, ensuring product decisions are conservative and don't overlook genuine issues.

About the JW Dataset

JW feedback is collected per-manuscript from screenshot-based forms. Items use fb_xxxx IDs (vs csv_fb_xxxx for IM/NN). Most entries lack agent_ratings and related_agents_label, but the free-text feedback often mentions specific agents (e.g. “Novelty Agent (very helpful)”). Completing sentiment labels would enable fuller integration with the main analysis.
