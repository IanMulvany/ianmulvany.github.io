Prepare a submission

Use this path if you need to register, choose a format, prepare files, enter metadata, submit, or replace a paper.

Apply a policy

Use this path when a volunteer or staff member needs a consistent decision flow for a submitted work.

  • Start with topical fit, scholarly standards, and rights to submit.
  • Return fixable format and metadata problems before declining content.
  • Escalate conduct, legal, authorship, and post-announcement concerns.
